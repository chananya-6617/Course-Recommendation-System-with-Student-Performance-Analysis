# Course-Recommendation-System-with-Student-Performance-Analysis

## Overview

The **Course Recommendation System with Student Performance Analysis** is a rule-based expert system developed using **Prolog**. The project analyzes a student's academic performance and interests to recommend suitable courses using logical inference and predefined rules.

Student information, performance details, and course data are stored as facts in the Prolog knowledge base. The system applies inference rules to evaluate the data and generate personalized course recommendations. This approach demonstrates the use of **Artificial Intelligence**, **knowledge representation**, and **logical reasoning** to support academic decision-making.

The primary objective of this project is to help students choose courses that align with their strengths, interests, and academic performance while showcasing the capabilities of Prolog in building intelligent, rule-based systems.

### Key Features

* Student performance analysis using Prolog facts and rules.
* Personalized course recommendations based on academic results.
* Rule-based decision-making without machine learning.
* Knowledge representation and logical inference using Prolog.
* Simple, explainable, and efficient recommendation system.

### Technologies Used

* **Programming Language:** Prolog
* **Platform:** SWI-Prolog
* **Concepts:** Artificial Intelligence, Knowledge Representation, Rule-Based Expert Systems, Logical Inference
